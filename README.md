# Kafka-Sensor-Streaming-System
Developed a real-time sensor telemetry streaming system using Apache Kafka and Python, implementing Kafka Producer and Consumer architecture for continuous live temperature data transmission, JSON-based messaging, and real-time monitoring of sensor data streams through Kafka topics.
Kafka Sensor Streaming System
Overview

This project demonstrates a real-time sensor telemetry streaming system using Apache Kafka and Python. A Kafka Producer continuously generates live temperature sensor data and sends it to a Kafka topic, while a Kafka Consumer receives and displays the streamed data in real time.
